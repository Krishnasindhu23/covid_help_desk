Of late, there has been growing anxiety on lack of beds in hospitals apart from that caused by covid itself. Taking this as the base, we have come up with a possible solution to tackle this situation through a web application called “ The Covid Help Desk.” Covid Help Desk is a dynamic web application which acts as an interface between persons in need of beds and hospitals. CHD provides the list of hospitals and the availability of beds in them. It allows the hospital department to update the changes in the number of beds time to time. We can also view the demand-supply gap state-wise. Aditionally, it also provides a list of measures & pratices for COVID home management.
